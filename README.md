# GAMEROOM 🎮 🕹️
Primer sistema <strong>Open Source</strong> para todo tipo de juegos




# Repositorio

```shell
git clone https://github.com/bemtorres/game-room
cd game-room

cp .env.example .env
composer install

npm install
npm run prod

php artisan migrate:fresh --seed

php artisan serve
```
